# 2027 年春节倒计时 · 迎接丁未新春

这是一个为“2027 年春节倒计时”特别打造的高品质、沉浸式前端网页。它不仅是一个精确到秒的计时器，更是一套完整的、充满新春氛围的视觉与交互体验。

## 预览

- **在线预览**: [https://1c7829eb1a25.aime-app.bytedance.net](https://1c7829eb1a25.aime-app.bytedance.net)

## ✨ 功能特性

- **精确倒计时**:
  - 目标锁定北京时间（Asia/Shanghai）**2027 年 2 月 6 日（星期六）00:00:00**（农历丁未年正月初一）。
  - 实时、精确地显示距离目标时刻的“天、时、分、秒”。
  - 倒计时结束后，页面将触发增强的烟花特效与“新春快乐”祝福弹幕。

- **沉浸式视觉与动效**:
  - **红金配色**: 页面采用经典的红金配色，营造浓厚的新春喜庆氛围。
  - **春节元素**: 融合了灯笼、福字、祥云、剪纸等多种传统春节视觉元素。
  - **轻盈动效**:
    - 背景中持续绽放的烟花粒子系统，为页面增添活力。
    - 悬挂的灯笼会随风轻微摇摆，祥云纹理缓慢流动。
    - 所有动效在保证美观的同时，也注重性能优化。

- **完整的页面结构**:
  - 告别单调的计时器，页面包含完整的首页结构，如 Hero 区域、特色说明、页脚等，提升了整体的品质感与信息完整度。

- **响应式与可访问性**:
  - 页面已为桌面和移动设备优化，确保在不同尺寸的屏幕上都有良好的阅读和视觉体验。
  - 核心倒计时区域在小屏幕上会居中并放大，保证信息可读性。
  - 尊重用户的系统设置，对开启“减少动态效果”（`prefers-reduced-motion`）的设备，将自动降低动画效果。
  - 为所有图片资源提供了明确的 `alt` 文本，提升了屏幕阅读器的可访问性。

## 🛠️ 技术栈

- **核心**: HTML5, CSS3, JavaScript (ES6+)
- **布局与样式**: [Tailwind CSS (CDN)](https://tailwindcss.com/)
- **字体**: [Google Fonts](https://fonts.google.com/) (Ma Shan Zheng, Noto Serif SC)
- **动效**: 原生 JavaScript + HTML Canvas, CSS Animations

## 📁 文件结构

项目遵循纯前端、单层目录的组织结构，所有资源均与入口文件 `index.html` 同级，以便于本地预览和部署。

```
spring-festival-2027-countdown/
│
├── 📄 index.html             # 页面主入口，包含所有 HTML 结构
├── 🎨 style.css              # 页面样式文件，包含所有自定义 CSS 与动画
├── ⚙️ script.js               # 核心逻辑文件，处理倒计时、烟花动效与交互
│
├── 🖼️ lantern-main.jpg        # 灯笼图片素材
├── 🖼️ fu-gold.png             # 金色“福”字图标
├── 🖼️ fu-papercut.jpg         # 剪纸风格“福”字装饰
├── 🖼️ clouds-gold.png         # 金色祥云纹样
│
└── 📖 README.md               # 项目说明文档（即本文）
```

## 🚀 本地运行

1.  克隆或下载此项目文件夹到本地。
2.  直接在浏览器中打开 `index.html` 文件即可预览。

> **注意**: 无需启动本地服务器，所有资源均通过相对路径引用，可直接在文件系统中运行。
